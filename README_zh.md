<h1>Breeze.JS</h1>
<h5>轻量而现代的 JavaScript 运行时</h5>
</div>

Breeze.JS 是一个基于 QuickJS 构建的轻量而现代的 JavaScript 运行时。它旨在提供一个简单高效的 JavaScript 代码运行环境。

## 特性

*   **轻量级:** 基于 QuickJS 引擎构建，以其小巧的体积而闻名。
*   **现代化:** 支持现代 JavaScript 特性。
*   **可嵌入:** 易于嵌入到 C/C++ 应用程序中。
*   **可扩展:** 提供机制通过原生模块扩展运行时。
